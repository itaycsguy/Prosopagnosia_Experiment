Visual_Recognition_Operations usage:
====================================
1) If this is the first download using this computer,run the file "install_packages.bat" using double-click.
**********************************************************************************************************************
2) Each time, run the file "get_visual_exp_tables.bat" using double-click.
This will produce:
- A folder called - visual_recognitions - next to the ".bat" files. It is containing folders that each contain all tables
- Each folder called "subject_tables_date_time_downloadNumber". It is containing all tables from the data-base.
(*) date + time is the current data the can demonstrate the current data usage.
**********************************************************************************************************************
